# felixriehm.de Pattern Lab

## What is it?

My personal website using Pattern Lab

## Which software features does it have?

* Bootstrap
* Written in HTML, CSS and JQuery
* SlickJS (image carousel)
* Atomic Design
* Pattern Lab
* Responsive
